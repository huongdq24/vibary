# **App Name**: Entremet Hanoi

## Core Features:

- Product Catalog: Display Entremet cakes with detailed descriptions, high-resolution images, and pricing.
- Flavor Quiz: Interactive quiz to recommend cakes based on user preferences using a tool with questions about preferred fruits, sweetness level, occasion, and personality traits.
- Shopping Cart: Enable users to add products to a shopping cart and manage quantities.
- Checkout Process: Handle the checkout process, including customer information, delivery address (Hanoi only), date/time selection, and payment method selection (Momo, ZaloPay, bank transfer, COD).
- News/Blog Articles: Provide articles about the company. Categories: founder stories, behind-the-scenes, annual reports, recruitment, shopping guides
- Contact Form and Hotline: Allow users to contact the shop with a form and phone number.
- Customer Reviews: Allow users to submit reviews and read other user reviews.

## Style Guidelines:

- Primary color: Pale pink (#F4DDDD) for a soft and inviting feel.
- Background color: Off-white (#F8F8F8) for a clean and minimalist aesthetic.
- Accent color: Soft gold (#E5C19F) for highlighting key elements and adding a touch of luxury.
- Headline font: 'Playfair', a modern sans-serif with high contrast lines, for an elegant, fashionable feel. Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth.
- Note: currently only Google Fonts are supported.
- Use delicate, custom-designed icons related to pastries, fruits, and Hanoi.
- Emphasize clean lines, generous white space, and a focus on high-quality imagery. Utilize full-width banners for featured collections and individual products.
- Incorporate subtle animations for smooth scrolling and interactive elements. Animate cake layers subtly on hover. Use CSS or JavaScript.
